Libreria di immagini di Visual Studio: azioni (formato .png, stile Windows Vista)

Immagini a 32 bit
Si tratta di un'immagine a 32 bit contenente un canale alfa per la trasparenza.

Limitazioni di utilizzo
Le immagini sono disponibili in varie dimensioni, sebbene nella tabella di seguito venga visualizzata una sola dimensione. Fare riferimento alla colonna "Dimensioni" per visualizzare le dimensioni disponibili. Vengono applicate le stesse restrizioni di utilizzo a tutte le dimensioni di una determinata immagine. In quanto elementi di un linguaggio visivo, le seguenti immagini, o qualsiasi parte di esse, devono essere utilizzate in maniera conforme alla descrizione di utilizzo fornita di seguito.
