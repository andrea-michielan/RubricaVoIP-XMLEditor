Libreria di immagini di Visual Studio: azioni (32 bit, formato .bmp, Strumenti dati)

Immagini a 32 bit
Si tratta di un'immagine a 32 bit contenente un canale alfa per la trasparenza.

Limitazioni di utilizzo
Le icone di azione vengono utilizzate per rappresentare i comandi nella struttura di menu. Nella maggior parte dei casi si tratta di verbi di azione, anche se a volte sono rappresentate da nomi (oggetti o strumenti) cui sono associate delle azioni, quali Mostra o Nascondi. Queste immagini sono comandi associati a Strumenti dati in Visual Studio.  In quanto elementi di un linguaggio visivo, le seguenti immagini, o qualsiasi parte di esse, devono essere utilizzate in maniera conforme agli Strumenti dati e alle indicazioni fornite di seguito.

Immagine Nome file Utilizzo
EntityDataModel_ADODotNETDataService.bmp ADO.NET Data Service
EntityDataModel_ComplexTypePropertyTool.bmp Strumento di proprietà di tipo complesso
EntityDataModel_OtherModelService.bmp Servizio di modello (generico)
