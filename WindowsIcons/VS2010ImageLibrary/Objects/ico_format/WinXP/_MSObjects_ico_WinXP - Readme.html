Libreria di immagini di Visual Studio: oggetti (formato .ico, stile Windows XP)

Immagine icona
Questo tipo di file può contenere più di una dimensione e/o profondità di colore di un'immagine. Le immagini icona vengono in genere utilizzate per controlli che rendono disponibili variazioni di stili di visualizzazione, ad esempio una casella di riepilogo che consente all'utente di scegliere tra "visualizzazione per icone piccole" e "visualizzazione per icone grandi". Nel controllo vengono visualizzate le dimensioni e la profondità di colore appropriate per l'immagine. Il modo migliore per visualizzare in anteprima i file delle icone è con Esplora risorse; nel browser, infatti, non vengono visualizzate le variazioni all'interno dei file.
