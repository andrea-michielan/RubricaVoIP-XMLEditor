Libreria di immagini di Visual Studio: animazioni

Limitazioni di utilizzo
In generale, le animazioni vengono utilizzate nelle finestre di dialogo Windows per indicare uno stato indeterminato, vale a dire lo stato per il cui completamento è richiesto un intervallo di tempo indeterminato. In quanto elementi di un linguaggio visivo, le immagini di questa cartella devono essere utilizzate in maniera conforme, sebbene non necessariamente in maniera identica, ai parametri di utilizzo elencati di seguito.
